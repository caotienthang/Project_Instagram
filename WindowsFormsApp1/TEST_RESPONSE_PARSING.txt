=== LƯU Ý QUAN TRỌNG ===

1. Authorization header này cần được gửi kèm trong mọi API requests sau khi login
2. Format: "Authorization: Bearer IGT:2:..."
3. SessionId format Instagram: {user_id}:{random}:{timestamp}:{hash}
4. CSRF token có thể không cần thiết cho mobile API (Bloks)
5. Response status = "ok" nghĩa là đăng nhập thành công
